# SOP: Session Logging

Sau mỗi session làm việc quan trọng, log lại theo format này.

## Format

```markdown
### [YYYY-MM-DD HH:MM] - [Tên task ngắn gọn]
- **Đã làm:** [Tóm tắt những gì đã hoàn thành]
- **Quyết định quan trọng:** [Lựa chọn kỹ thuật/design đã thống nhất]
- **Vấn đề đang mở:** [Còn dở dang hoặc cần làm tiếp]
- **Context cần nhớ:** [Thông tin quan trọng cho session sau]
```

## Lưu vào

`_memory-logs/YYMM-log.md` - một file per tháng.

Ví dụ: `_memory-logs/2605-log.md` cho tháng 5/2026.

## Khi nào log

- Sau khi hoàn thành một feature/task lớn
- Khi sắp kết thúc session
- Khi gặp quyết định quan trọng
- Trước khi chuyển sang task khác hoàn toàn
